# API

When you create a Corsair instance using `createCorsair`, you are provided with every endpoint that exists across all of your plug-ins. 

Any new plugin added to Corsair will be accessible through the Corsair instance. 

## Calling API endpoints


```ts title="corsair.ts"
import { corsair } from "corsair";

export const corsair = createCorsair({
    //...
})

// calling a plugin
await corsair.slack.api.messages.post({
    channel: "abc_123",
    text: "Hello, world!"
})
```
