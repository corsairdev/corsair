## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/fireflies
```

Add the Fireflies plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { fireflies } from "@corsair-dev/fireflies";

export const corsair = createCorsair({
    plugins: [fireflies()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { fireflies } from "@corsair-dev/fireflies";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [fireflies()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Fireflies API key

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=fireflies api_key=your-api-key
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=fireflies webhook_signature=your-webhook-secret
```

See [Get Credentials](/plugins/fireflies/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --plugin=fireflies api_key=their-api-key --tenant=user_abc123
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=fireflies webhook_signature=your-webhook-secret
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.


## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | API key (optional, uses database if not provided) |
| `webhookSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// List transcripts
const transcripts = await corsair.fireflies.api.transcripts.list({});

// Get a specific transcript
const transcript = await corsair.fireflies.api.transcripts.get({
    id: "transcript-id",
});

// Get current user
const user = await corsair.fireflies.api.users.get({});

// Ask Fireflies AI about a transcript
const answer = await corsair.fireflies.api.askFred.ask({
    transcriptId: "transcript-id",
    question: "What were the main action items?",
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user-123");

// List transcripts
const transcripts = await tenant.fireflies.api.transcripts.list({});

// Get a specific transcript
const transcript = await tenant.fireflies.api.transcripts.get({

const user = await tenant.fireflies.api.users.get({});

// Ask Fireflies AI about a transcript
const answer = await tenant.fireflies.api.askFred.ask({
    transcriptId: "transcript-id",
    question: "What were the main action items?",
});
```
</Tab>
</Tabs>
