## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/stripe
```

Add the Stripe plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { stripe } from "@corsair-dev/stripe";

export const corsair = createCorsair({
    plugins: [stripe()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { stripe } from "@corsair-dev/stripe";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [stripe()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Stripe secret key

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=stripe api_key=sk_live_your-secret-key
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=stripe webhook_signature=whsec_your-webhook-secret
```

See [Get Credentials](/plugins/stripe/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
Enable `multiTenancy` in your Corsair config, then store each user's Stripe key separately:

```ts corsair.ts
export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [stripe()],
    kek: process.env.CORSAIR_KEK!,
});
```

```bash
pnpm corsair setup --plugin=stripe api_key=sk_live_their-key --tenant=user_abc123
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=stripe webhook_signature=whsec_your-webhook-secret
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | Secret key (optional, uses database if not provided) |
| `webhookSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// Get account balance
const balance = await corsair.stripe.api.balance.get({});

// Create a customer
const customer = await corsair.stripe.api.customers.create({
    email: "user@example.com",
    name: "Jane Doe",
});

// Create a payment intent
const intent = await corsair.stripe.api.paymentIntents.create({
    amount: 2000,
    currency: "usd",
    customer: customer.id,
});

// List charges
const charges = await corsair.stripe.api.charges.list({
    limit: 20,
});

// Create a price
const price = await corsair.stripe.api.prices.create({
    unit_amount: 1999,
    currency: "usd",
    recurring: { interval: "month" },
    product: "prod-id",
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("account-123");

// Get account balance
const balance = await tenant.stripe.api.balance.get({});

// Create a customer
const customer = await tenant.stripe.api.customers.create({
    email: "user@example.com",
    name: "Jane Doe",
});

// Create a payment intent
const intent = await tenant.stripe.api.paymentIntents.create({
    amount: 2000,
    currency: "usd",
    customer: customer.id,
});

// List charges
const charges = await tenant.stripe.api.charges.list({
    limit: 20,
});
```
</Tab>
</Tabs>
