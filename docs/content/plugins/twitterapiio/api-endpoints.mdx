The TwitterApiIO plugin provides access to Twitter's API via twitterapi.io for tweets, users, lists, communities, and trends.

**New to Corsair?** Learn about core concepts like [API access](/concepts/api), [authentication](/concepts/auth), and [error handling](/concepts/error-handling).
</Info>

<Note>
**Full Implementation:** For the complete, up-to-date list, see the [TwitterApiIO plugin source code](https://github.com/corsairdev/corsair/tree/main/packages/corsair/plugins/twitterapiio).
</Note>

## Tweets

### search
`tweets.search`

Search tweets with a query.

```ts
const tweets = await corsair.twitterapiio.api.tweets.search({
    query: "corsair framework", // [!code highlight]
    maxResults: 20,
});
```

**Parameters:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `query` | `string` | Yes | Twitter search query |
| `maxResults` | `number` | No | Max results to return |

## Trends

### get
`trends.get`

Get trending topics by location (WOEID).

```ts
const trends = await corsair.twitterapiio.api.trends.get({
    woeid: 1, // [!code highlight] 1 = worldwide
});
```

**Parameters:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `woeid` | `number` | Yes | Where On Earth ID (1 = worldwide) |
