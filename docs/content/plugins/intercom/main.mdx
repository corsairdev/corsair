## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/intercom
```

Add the Intercom plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { intercom } from "@corsair-dev/intercom";

export const corsair = createCorsair({
    plugins: [intercom()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { intercom } from "@corsair-dev/intercom";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [intercom()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Access token

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --intercom api_key=your-access-token
```

For webhook signature verification:

```bash
pnpm corsair setup --intercom webhook_signature=your-client-secret
```

See [Get Credentials](/plugins/intercom/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
Enable `multiTenancy` in your Corsair config, then store each user's key separately:

```ts corsair.ts
export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [intercom()],
    kek: process.env.CORSAIR_KEK!,
});
```

```bash
pnpm corsair setup --intercom api_key=their-access-token --tenant=user_abc123
```

For webhook signature verification:

```bash
pnpm corsair setup --intercom webhook_signature=your-client-secret
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | Access token (optional, uses database if not provided) |
| `webhookSecret` | `string` | Client secret for webhook verification (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// List contacts
const contacts = await corsair.intercom.api.contacts.list({});

// Search contacts
const results = await corsair.intercom.api.contacts.search({
    query: {
        field: "email",
        operator: "=",
        value: "user@example.com",
    },
});

// List open conversations
const conversations = await corsair.intercom.api.conversations.list({
    open: true,
});

// Reply to a conversation
await corsair.intercom.api.conversations.reply({
    conversationId: "conv-id",
    body: "Thanks for reaching out!",
    type: "admin",
});

// Create an article
await corsair.intercom.api.articles.create({
    title: "Getting Started",
    body: "<p>Welcome to our product...</p>",
    state: "published",
});
```
</Tab>
<Tab title="Multi-Tenant">

</Tab>
