## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/hubspot
```

Add the HubSpot plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { hubspot } from "@corsair-dev/hubspot";

export const corsair = createCorsair({
    plugins: [
        hubspot(),
    ],
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { hubspot } from "@corsair-dev/hubspot";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [
        hubspot(),
    ],
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

Once configured, you can access the HubSpot API through your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts
const contacts = await corsair.hubspot.api.contacts.getMany({
    limit: 10,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user_id");

const contacts = await tenant.hubspot.api.contacts.getMany({
    limit: 10,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

The HubSpot plugin supports both API key and OAuth 2.0 authentication methods.

- **[`api_key`](/content/concepts/api-key)** - Private App API key authentication
- **[`oauth_2`](/content/concepts/oauth)** - OAuth 2.0 authentication

### API Key Authentication (Private App)

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=hubspot api_key=your-api-key
```

See [Get Credentials](/plugins/hubspot/get-credentials) for step-by-step instructions on obtaining an API key.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --plugin=hubspot api_key=their-api-key --tenant=user_abc123
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

### OAuth 2.0 Authentication

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=hubspot client_id=your-client-id client_secret=your-client-secret
pnpm corsair auth --plugin=hubspot
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=hubspot webhook_signature=your-webhook-secret
```

See [Get Credentials](/plugins/hubspot/get-credentials) for step-by-step instructions on setting up OAuth 2.0.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --plugin=hubspot client_id=your-client-id client_secret=your-client-secret
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=hubspot webhook_signature=your-webhook-secret
```
</Tab>
</Tabs>

See [OAuth 2.0 Authentication](/content/concepts/oauth) for full setup including callback handling.

## Options

The HubSpot plugin accepts the following configuration options:

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key' \| 'oauth_2'` | Authentication method |
| `credentials` | `object` | Credentials object (apiKey for api_key, clientId/clientSecret for oauth_2) |
| `webhookSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

For complete documentation on getting credentials, see the [Plugin Credentials Guide](/guides/plugin-credentials#hubspot).
