title: Calendly

---

## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/calendly
```

Add the Calendly plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { calendly } from "@corsair-dev/calendly";

export const corsair = createCorsair({
    plugins: [calendly()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { calendly } from "@corsair-dev/calendly";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [calendly()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Personal access token

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=calendly api_key=your-personal-access-token
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=calendly webhook_signature=your-signing-key
```

See [Get Credentials](/plugins/calendly/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --plugin=calendly api_key=their-api-key --tenant=user_abc123
```

For webhook signature verification:

```bash
pnpm corsair setup --plugin=calendly webhook_signature=your-signing-key
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | API key (optional, uses database if not provided) |
| `webhookSecret` | `string` | Webhook signing key (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// Get current user
const me = await corsair.calendly.api.users.getCurrent({});

// List scheduled events
const events = await corsair.calendly.api.scheduledEvents.list({
    organization: "https://api.calendly.com/organizations/org-id",
});

// List event types
const eventTypes = await corsair.calendly.api.eventTypes.list({
    user: "https://api.calendly.com/users/user-id",
});

// Get invitees for an event
const invitees = await corsair.calendly.api.invitees.list({
    uuid: "event-uuid",
});

// Create a scheduling link
const link = await corsair.calendly.api.schedulingLinks.create({
    max_event_count: 1,
    owner: "https://api.calendly.com/event_types/type-id",
    owner_type: "EventType",
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user-123");

// Get current user
const me = await tenant.calendly.api.users.getCurrent({});

// List scheduled events
const events = await tenant.calendly.api.scheduledEvents.list({
    organization: "https://api.calendly.com/organizations/org-id",
});

// List event types
const eventTypes = await tenant.calendly.api.eventTypes.list({
    user: "https://api.calendly.com/users/user-id",
});
```
</Tab>
</Tabs>
