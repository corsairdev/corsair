## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/cursor
```

Add the Cursor plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { cursor } from "@corsair-dev/cursor";

export const corsair = createCorsair({
    plugins: [cursor()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { cursor } from "@corsair-dev/cursor";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [cursor()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Cursor API key

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --cursor api_key=your-api-key
```

See [Get Credentials](/plugins/cursor/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --cursor api_key=their-api-key --tenant=user_abc123
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | API key (optional, uses database if not provided) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// List agents
const agents = await corsair.cursor.api.agents.list({
    limit: 20,
});

// Get agent conversation
const conversation = await corsair.cursor.api.agents.getConversation({
    id: "agent-id",
});

// Get account info
const me = await corsair.cursor.api.account.getMe({});

// List available models
const models = await corsair.cursor.api.models.list({});

// List repositories
const repos = await corsair.cursor.api.repositories.list({});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user-123");

// List agents
const agents = await tenant.cursor.api.agents.list({
    limit: 20,
});

// Get agent conversation
const conversation = await tenant.cursor.api.agents.getConversation({
    id: "agent-id",
});

// Get account info
const me = await tenant.cursor.api.account.getMe({});
```
</Tab>
</Tabs>
