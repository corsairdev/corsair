## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/exa
```

Add the Exa plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { exa } from "@corsair-dev/exa";

export const corsair = createCorsair({
    plugins: [exa()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { exa } from "@corsair-dev/exa";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [exa()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Exa API key

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=exa api_key=your-api-key
```

See [Get Credentials](/plugins/exa/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --plugin=exa api_key=their-api-key --tenant=user_abc123
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | API key (optional, uses database if not provided) |
| `webhookSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// Neural search
const results = await corsair.exa.api.search.search({
    query: "latest AI research 2025",
    numResults: 10,
});

// Get page contents
const contents = await corsair.exa.api.contents.get({
    ids: ["https://example.com/article"],
});

// Ask a question with web grounding
const answer = await corsair.exa.api.answer.answer({
    query: "What is the latest Claude model?",
});

// Create a webset for monitoring
const webset = await corsair.exa.api.websets.create({
    search: { query: "AI startups", count: 20 },
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user-123");

// Neural search
const results = await tenant.exa.api.search.search({
    query: "latest AI research 2025",
    numResults: 10,
});

// Get page contents
const contents = await tenant.exa.api.contents.get({
    ids: ["https://example.com/article"],
});
```
</Tab>
</Tabs>
