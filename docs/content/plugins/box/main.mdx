## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/box
```

Add the Box plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { box } from "@corsair-dev/box";

export const corsair = createCorsair({
    plugins: [box()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { box } from "@corsair-dev/box";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [box()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`oauth_2`](/content/concepts/oauth)** (default) - OAuth 2.0 user authentication

### OAuth 2.0 Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --box client_id=your-client-id client_secret=your-client-secret
pnpm corsair auth --plugin=box
```

See [Get Credentials](/plugins/box/get-credentials) for step-by-step instructions on setting up OAuth 2.0.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --box client_id=your-client-id client_secret=your-client-secret
```
</Tab>
</Tabs>

See [OAuth 2.0 Authentication](/content/concepts/oauth) for full setup including callback handling.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'oauth_2'` | Authentication method |
| `webhookSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// List files in a folder (root = "0")
const files = await corsair.box.api.files.list({
    folderId: "0",
});

// Get a specific file
const file = await corsair.box.api.files.get({
    fileId: "file-id",
});

// Create a folder
await corsair.box.api.folders.create({
    name: "New Folder",
    parentId: "0",
});

// Upload a file
await corsair.box.api.files.upload({
    name: "document.pdf",
    parentId: "0",
    content: fileBuffer,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user-123");

// List files in a folder (root = "0")
const files = await tenant.box.api.files.list({
    folderId: "0",
});

// Get a specific file
const file = await tenant.box.api.files.get({
    fileId: "file-id",
});

// Create a folder
await tenant.box.api.folders.create({
    name: "New Folder",
    parentId: "0",
});
```
</Tab>
</Tabs>
