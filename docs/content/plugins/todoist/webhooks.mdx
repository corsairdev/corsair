The Todoist plugin includes webhook handlers for Todoist task, note, and project lifecycle events.

<Callout type="info">
**New to Corsair?** Learn about [webhooks](/concepts/webhooks) and [hooks](/concepts/hooks).
</Callout>

<Callout>
**Full Implementation:** See the [Todoist plugin source code](https://github.com/corsairdev/corsair/tree/main/packages/corsair/plugins/todoist).
</Callout>

## Setup

```ts title="app/api/webhook/route.ts"
import { processWebhook } from "corsair";
import { corsair } from "@/server/corsair";

export async function POST(request: Request) {
    const headers = Object.fromEntries(request.headers);
    const body = await request.json();

    const result = await processWebhook(corsair, headers, body);
    return result.response;
}
```

Todoist webhooks are identified by the `x-todoist-delivery-id` header.

## Available Webhooks

### items.added

Fires when a new task is added.

**Example:**

```ts title="corsair.ts"
todoist({
    webhookHooks: {
        "items.added": async (event) => {
            console.log("Task added:", event.event_data?.content);
        },
    },
})
```

**Key fields:** `event_data.id`, `event_data.content`, `event_data.project_id`, `event_data.due`, `event_data.priority`, `initiator`
