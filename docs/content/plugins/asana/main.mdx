## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/asana
```

Add the Asana plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { asana } from "@corsair-dev/asana";

export const corsair = createCorsair({
    plugins: [asana()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { asana } from "@corsair-dev/asana";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [asana()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **`api_key`** (default) - Personal access token

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --asana api_key=your-personal-access-token
```

See [Get Credentials](/plugins/asana/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --asana api_key=their-api-key --tenant=user_abc123
```
</Tab>
</Tabs>

See [API Key Authentication](/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method |
| `key` | `string` | API key (optional, uses database if not provided) |
| `webhookSecret` | `string` | Webhook secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// List tasks in a project
const tasks = await corsair.asana.api.projects.getTasks({
    project_gid: "project-id",
});

// Create a task
await corsair.asana.api.tasks.create({
    name: "Fix login bug",
    projects: ["project-id"],
    notes: "Users can't log in with SSO",
    due_on: "2024-01-15",
});

// Update a task
await corsair.asana.api.tasks.update({
    task_gid: "task-id",
    completed: true,
});

// List projects
const projects = await corsair.asana.api.projects.list({
    workspace: "workspace-id",
});

// Get current user
const me = await corsair.asana.api.users.getCurrent({});

// List team members
const members = await corsair.asana.api.teams.get({
    team_gid: "team-id",
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("user-123");

// List tasks in a project
const tasks = await tenant.asana.api.projects.getTasks({
    project_gid: "project-id",
});

// Create a task
await tenant.asana.api.tasks.create({
    name: "Fix login bug",
    projects: ["project-id"],
    notes: "Users can't log in with SSO",
    due_on: "2024-01-15",
});

// Update a task
await tenant.asana.api.tasks.update({
    task_gid: "task-id",
    completed: true,
});
```
</Tab>
</Tabs>
