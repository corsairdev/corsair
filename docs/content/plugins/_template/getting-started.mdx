## Setup

<Steps>

<Step title="Install the plugin">

<CodeGroup>
```bash npm
npm install corsair @corsair-dev/{{PLUGIN_PACKAGE_NAME}}
```
```bash yarn
yarn add corsair @corsair-dev/{{PLUGIN_PACKAGE_NAME}}
```
```bash pnpm
pnpm install corsair @corsair-dev/{{PLUGIN_PACKAGE_NAME}}
```
```bash bun
bun add corsair @corsair-dev/{{PLUGIN_PACKAGE_NAME}}
```
</CodeGroup>

</Step>

<Step title="Add to corsair.ts">

<Tabs>
<Tab title="SQLite">

```ts src/server/corsair.ts
import 'dotenv/config';
import Database from 'better-sqlite3';
import { createCorsair } from 'corsair';
import { {{PLUGIN_IMPORT_NAME}} } from '@corsair-dev/{{PLUGIN_PACKAGE_NAME}}';

const db = new Database('corsair.db');

export const corsair = createCorsair({
    plugins: [{{PLUGIN_IMPORT_NAME}}()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```

</Tab>
<Tab title="PostgreSQL">

```ts src/server/corsair.ts
import 'dotenv/config';
import { Pool } from 'pg';
import { createCorsair } from 'corsair';
import { {{PLUGIN_IMPORT_NAME}} } from '@corsair-dev/{{PLUGIN_PACKAGE_NAME}}';

const db = new Pool({ connectionString: process.env.DATABASE_URL });

export const corsair = createCorsair({
    plugins: [{{PLUGIN_IMPORT_NAME}}()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```

</Tab>
</Tabs>

</Step>

## Multi-tenancy

Building a product? Scope all calls to a tenant with a single line:

```ts

// API calls and DB queries are isolated to this tenant
const {{PLUGIN_RESOURCE_PLURAL}} = await tenant.{{PLUGIN_CLI_NAME}}.api.{{PLUGIN_RESOURCE_GROUP}}.list({});
```
