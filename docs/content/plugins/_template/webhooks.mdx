Corsair normalizes {{PLUGIN_DISPLAY_NAME}}'s webhook payloads and routes them to your handlers automatically. Register a webhook endpoint and subscribe to any events below.

<Info>
**New to webhooks?** See the [Webhooks guide](/guides/webhooks) to set up ngrok and register your endpoint first.
</Info>

## Register your endpoint

```ts src/server/corsair.ts
{{PLUGIN_IMPORT_NAME}}({
    webhookSecret: process.env.{{PLUGIN_WEBHOOK_SECRET_VAR}},
})
```

Mount the handler in your app:

```ts
// Express / Next.js API route
app.post('/webhooks/{{PLUGIN_CLI_NAME}}', express.raw({ type: '*/*' }), async (req, res) => {
    await corsair.{{PLUGIN_CLI_NAME}}.handleWebhook(req.body, req.headers);
    res.sendStatus(200);
});
```

## Webhook hooks

Run custom logic before or after Corsair processes an event:

```ts
{{PLUGIN_IMPORT_NAME}}({
    webhookHooks: {
        {{RESOURCE_GROUP_1_KEY}}: {
            {{EVENT_HOOK_NAME}}: {
                before: async (ctx, payload) => {
                    // runs before Corsair syncs the event to the database
                    console.log('incoming event:', payload);
                },
                after: async (ctx, result) => {
                    // runs after Corsair has synced the event
                    console.log('synced entity:', result.data);
                },
            },
        },
    },
})
```
