## Quick Start

Install the plugin:

```bash
pnpm install @corsair-dev/monday
```

Add the Monday.com plugin to your Corsair instance:

<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from "corsair";
import { monday } from "@corsair-dev/monday";

export const corsair = createCorsair({
    plugins: [monday()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from "corsair";
import { monday } from "@corsair-dev/monday";

export const corsair = createCorsair({
    multiTenancy: true,
    plugins: [monday()],
    database: db,
    kek: process.env.CORSAIR_KEK!,
});
```
</Tab>
</Tabs>

## Authentication

### Supported Auth Types

- **[`api_key`](/content/concepts/api-key)** (default) - Personal API token

### API Key Setup

<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --monday api_key=your-api-token
```

For webhook signature verification:

```bash
pnpm corsair setup --monday webhook_signature=your-webhook-secret
```

See [Get Credentials](/plugins/monday/get-credentials) for step-by-step instructions.
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --monday api_key=their-api-token --tenant=user_abc123
```

For webhook signature verification:

```bash
pnpm corsair setup --monday webhook_signature=your-webhook-secret
```
</Tab>
</Tabs>

See [API Key Authentication](/content/concepts/api-key) for full setup details.

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `'api_key'` | Authentication method (defaults to `'api_key'`) |
| `key` | `string` | API token (optional, uses database if not provided) |
| `webhookSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | Endpoint hooks for custom logic |
| `webhookHooks` | `object` | Webhook hooks for event handling |
| `errorHandlers` | `object` | Custom error handlers |

## Usage

<Tabs>
<Tab title="Solo">
```ts
// List boards
const boards = await corsair.monday.api.boards.list({});

// Get board items
const items = await corsair.monday.api.items.list({
    boardId: "board-id",
});

// Create an item
await corsair.monday.api.items.create({
    boardId: "board-id",
    groupId: "group-id",
    itemName: "New Task",
});

// Add an update (comment) to an item
await corsair.monday.api.updates.create({
    itemId: "item-id",
    body: "Status updated",
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts
const tenant = corsair.withTenant("org-123");

// List boards
const boards = await tenant.monday.api.boards.list({});

// Get board items
const items = await tenant.monday.api.items.list({
    boardId: "board-id",
});

// Create an item
await tenant.monday.api.items.create({
    boardId: "board-id",
    groupId: "group-id",
    itemName: "New Task",
});

// Add an update (comment) to an item
await tenant.monday.api.updates.create({
    itemId: "item-id",
    body: "Status updated",
});
```
</Tab>
</Tabs>
