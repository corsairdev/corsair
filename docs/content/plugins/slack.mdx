Slack integration for Corsair. Build powerful Slack automations with full API access, real-time webhooks, and automatic data synchronization.

## Installation

Add the plugin to your Corsair instance:

```ts title="corsair.ts"
import { createCorsair } from "corsair";
import { slack } from "corsair/plugins";

export const corsair = createCorsair({
    plugins: [
        slack({
            authType: "api_key",
            credentials: {
                botToken: process.env.SLACK_BOT_TOKEN,
                signingSecret: process.env.SLACK_SIGNING_SECRET,
            },
        }),
    ],
});
```

## Authentication

Slack supports the following auth types:

### API Key

Use a bot token from your Slack app settings:

```ts title="corsair.ts"
slack({
    authType: "api_key",
    credentials: {
        botToken: process.env.SLACK_BOT_TOKEN,
        signingSecret: process.env.SLACK_SIGNING_SECRET, // Optional, for webhook verification
    },
})
```

### OAuth 2.0

```ts title="corsair.ts"
slack({
    authType: "oauth_2",
    credentials: {
        clientId: process.env.SLACK_CLIENT_ID,
        clientSecret: process.env.SLACK_CLIENT_SECRET,
    },
})
```

## Services

Slack provides the following services:

| Service | Description |
|---------|-------------|
| `channels` | Manage channels, conversations, and direct messages |
| `users` | Access user profiles, presence, and information |
| `userGroups` | Create and manage user groups |
| `files` | Upload, list, and retrieve files |
| `messages` | Send, update, delete, and search messages |
| `reactions` | Add and remove emoji reactions |
| `stars` | Star and unstar items |

## Webhooks

Slack supports the following webhook events:

| Event | Description |
|-------|-------------|
| `message` | New message posted in a channel |
| `channelCreated` | New channel created |
| `reactionAdded` | Reaction added to a message |
| `teamJoin` | New user joined the workspace |
| `userChange` | User profile updated |
| `fileCreated` | New file uploaded |
| `filePublic` | File made public |
| `fileShared` | File shared to a channel |

### Handling Webhooks

```ts title="corsair.ts"
slack({
    authType: "api_key",
    credentials: {
        botToken: process.env.SLACK_BOT_TOKEN,
        signingSecret: process.env.SLACK_SIGNING_SECRET,
    },
    webhookHooks: {
        messages: {
            message: {
                after: async (ctx, result) => {
                    console.log("New message:", result);
                },
            },
        },
        channels: {
            created: {
                after: async (ctx, result) => {
                    console.log("Channel created:", result);
                },
            },
        },
        reactions: {
            added: {
                after: async (ctx, result) => {
                    console.log("Reaction added:", result);
                },
            },
        },
        users: {
            teamJoin: {
                after: async (ctx, result) => {
                    console.log("User joined:", result);
                },
            },
            userChange: {
                after: async (ctx, result) => {
                    console.log("User updated:", result);
                },
            },
        },
        files: {
            created: {
                after: async (ctx, result) => {
                    console.log("File created:", result);
                },
            },
            shared: {
                after: async (ctx, result) => {
                    console.log("File shared:", result);
                },
            },
        },
    },
})
```

## Hooks

Add before/after hooks to customize behavior.

### API Hooks

```ts title="corsair.ts"
slack({
    authType: "api_key",
    credentials: { ... },
    hooks: {
        messages: {
            post: {
                before: (ctx, args) => {
                    console.log("Before posting message:", args);
                    return { ctx, args };
                },
                after: (ctx, result) => {
                    console.log("Message posted:", result.ts);
                },
            },
        },
        channels: {
            create: {
                before: (ctx, args) => {
                    // Enforce naming convention
                    args.name = args.name.toLowerCase().replace(/\s+/g, "-");
                    return { ctx, args };
                },
                after: (ctx, result) => {
                    console.log("Channel created:", result.channel?.id);
                },
            },
        },
    },
})
```

### Webhook Hooks

```ts title="corsair.ts"
slack({
    authType: "api_key",
    credentials: { ... },
    webhookHooks: {
        messages: {
            message: {
                before: async (ctx, payload) => {
                    // Filter or transform incoming webhooks
                    return { ctx, payload };
                },
                after: async (ctx, result) => {
                    // Sync to external system, send notification, etc.
                },
            },
        },
    },
})
```

## Error Handling

The Slack plugin includes built-in error handlers for common scenarios:

| Error Type | Matches | Behavior |
|------------|---------|----------|
| `RATE_LIMIT_ERROR` | `rate_limited`, `429` | Retries up to 5 times with backoff |
| `AUTH_ERROR` | `invalid_auth`, `token_revoked`, `token_expired`, `not_authed` | No retry (fix credentials) |
| `NOT_FOUND_ERROR` | `channel_not_found`, `user_not_found`, `file_not_found`, `message_not_found` | No retry |

Add custom error handlers:

```ts title="corsair.ts"
slack({
    authType: "api_key",
    credentials: { ... },
    errorHandlers: {
        RATE_LIMIT_ERROR: {
            match: (error) => error.message.includes("rate_limited"),
            handler: async (error, context) => ({
                maxRetries: 5,
                retryStrategy: "exponential_backoff_jitter",
            }),
        },
        PERMISSION_ERROR: {
            match: (error) => error.message.includes("missing_scope"),
            handler: async (error, context) => {
                console.error("Missing Slack scope:", error.message);
                return { maxRetries: 0 };
            },
        },
        DEFAULT: {
            match: () => true,
            handler: async (error, context) => {
                console.error(`Slack error:`, error.message);
                return { maxRetries: 0 };
            },
        },
    },
})
```

## Database Schema

Corsair stores Slack data in the `corsair_resources` table. Each resource type has a strongly-typed schema.

### Querying Data

```ts title="example.ts"
// Find by resource ID
const channel = await corsair.slack.db.channels.findByResourceId("C0123456789");

// List all
const channels = await corsair.slack.db.channels.list({ limit: 50 });

// Search
const results = await corsair.slack.db.messages.search({
    query: "project update",
    limit: 20,
});

// Upsert
await corsair.slack.db.users.upsert("U0123456789", {
    id: "U0123456789",
    name: "johndoe",
    real_name: "John Doe",
    email: "john@example.com",
});
```

| `is_usergroup` | `boolean` | Is a user group |
| `date_create` | `number` | Creation timestamp |
| `date_update` | `number` | Last update timestamp |
| `created_by` | `string` | Creator user ID |
| `users` | `string[]` | Member user IDs |
| `user_count` | `number` | Number of users |
| `channel_count` | `number` | Number of channels |

## Options

| Option | Type | Description |
|--------|------|-------------|
| `authType` | `"api_key"` \| `"oauth_2"` | Authentication method |
| `credentials` | `object` | Authentication credentials |
| `credentials.botToken` | `string` | Slack bot token (for API key auth) |
| `credentials.signingSecret` | `string` | Webhook signing secret (optional) |
| `hooks` | `object` | API hooks for before/after actions |
| `webhookHooks` | `object` | Webhook hooks for before/after events |
| `errorHandlers` | `object` | Custom error handlers |

## Full Example

```ts title="corsair.ts"
import { createCorsair } from "corsair";
import { drizzleAdapter } from "corsair/adapters/drizzle";
import { slack } from "corsair/plugins";

export const corsair = createCorsair({
    multiTenancy: true,
    database: drizzleAdapter(db, { provider: "pg", schema }),
    plugins: [
        slack({
            authType: "api_key",
            credentials: {
                botToken: process.env.SLACK_BOT_TOKEN,
                signingSecret: process.env.SLACK_SIGNING_SECRET,
            },
            hooks: {
                messages: {
                    post: {
                        after: (ctx, result) => {
                            console.log("Message sent:", result.ts);
                        },
                    },
                },
                channels: {
                    create: {
                        before: (ctx, args) => {
                            args.name = args.name.toLowerCase();
                            return { ctx, args };
                        },
                    },
                },
            },
            webhookHooks: {
                messages: {
                    message: {
                        after: async (ctx, result) => {
                            // Auto-respond to messages mentioning the bot
                            console.log("New message received");
                        },
                    },
                },
            },
            errorHandlers: {
                RATE_LIMIT_ERROR: {
                    match: (error) => error.message.includes("rate"),
                    handler: async () => ({
                        maxRetries: 3,
                        retryStrategy: "exponential_backoff_jitter",
                    }),
                },
            },
        }),
    ],
});
```
