## Profile

### get

`profile.get`

Get the authenticated user Oura profile and personal info

**Risk:** `read`

```ts
await corsair.oura.api.profile.get({});
```

**Input:** _empty object_


**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | — |
| `age` | `number` | No | — |
| `weight` | `number` | No | — |
| `height` | `number` | No | — |
| `biological_sex` | `string` | No | — |
| `email` | `string` | No | — |
