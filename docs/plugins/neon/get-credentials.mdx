## API key

Use a Neon API key for all operations.

1. Open the Neon console.
2. Go to **Account settings** → **API keys**.
3. Create a new API key.
4. Store it as `api_key`.

```bash
pnpm corsair setup --neon api_key=<neon_api_key>
```

Personal API keys have access to every project your account can reach. For organization-scoped access, create an organization API key instead under **Organization settings** → **API keys** and store it the same way.
