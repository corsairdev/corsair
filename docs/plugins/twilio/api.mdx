## Calls

### get

`calls.get`

**Risk:** `read`

```ts
await corsair.twilio.api.calls.get({});
```

### list

`calls.list`

List Twilio calls for an account

**Risk:** `read`

```ts
await corsair.twilio.api.calls.list({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `limit` | `number` | No | — |
| `to` | `string` | No | — |
| `from` | `string` | No | — |
| `status` | `queued \| ringing \| in-progress \| canceled \| completed \| busy \| failed \| no-answer` | No | — |
| `startTimeAfter` | `string` | No | — |
| `startTimeBefore` | `string` | No | — |
| `endTimeAfter` | `string` | No | — |
| `endTimeBefore` | `string` | No | — |
| `pageSize` | `number` | No | — |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `calls` | `object[]` | Yes | — |
| `end` | `number` | No | — |
| `first_page_uri` | `string` | No | — |
| `next_page_uri` | `string` | No | — |
| `page` | `number` | No | — |
| `page_size` | `number` | No | — |
| `previous_page_uri` | `string` | No | — |
| `start` | `number` | No | — |
| `uri` | `string` | No | — |
