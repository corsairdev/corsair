## Account

### getMe

`account.getMe`

Retrieve API key information including name, creation date, and owner email

**Risk:** `read`

```ts
await corsair.cursor.api.account.getMe({});
```

**Input:** _empty object_


**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `apiKeyName` | `string` | Yes | ‚Äî |
| `createdAt` | `string` | Yes | ‚Äî |
| `userEmail` | `string` | No | ‚Äî |

## Agents

### getConversation

`agents.getConversation`

Retrieve the conversation history for a specific cloud agent

**Risk:** `read`

```ts
await corsair.cursor.api.agents.getConversation({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | ‚Äî |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | ‚Äî |
| `messages` | `object[]` | Yes | ‚Äî |

<AccordionGroup>
<Accordion title="messages full type">

```ts
{
  id?: string,
```

</Accordion>
</AccordionGroup>

### list

`agents.list`

Retrieve a paginated list of Cursor Cloud agents

**Risk:** `read`

```ts
await corsair.cursor.api.agents.list({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `limit` | `number` | No | ‚Äî |
| `cursor` | `string` | No | ‚Äî |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `agents` | `object[]` | Yes | ‚Äî |
| `nextCursor` | `string` | No | ‚Äî |

<AccordionGroup>
<Accordion title="agents full type">

```ts
{
  id?: string,
  name?: string,
  source?: {
    ref?: string,
    repository?: string

  target?: {
    url?: string,
    prUrl?: string,
    branchName?: string,
    autoCreatePr?: boolean,
    skipReviewerRequest?: boolean,
    openAsCursorGithubApp?: boolean
  },
  summary?: string,
  createdAt?: string
}[]
```
</Accordion>
</AccordionGroup>
