## Answer

### get

`answer.get`

Generate a direct, citation-backed answer to a natural language question

**Risk:** `read`

```ts
await corsair.exa.api.answer.get({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `query` | `string` | Yes | — |
| `text` | `boolean` | No | — |
| `stream` | `boolean` | No | — |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `answer` | `string` | Yes | — |
| `citations` | `object[]` | No | — |
| `requestId` | `string` | No | — |

<AccordionGroup>
<Accordion title="citations full type">

```ts
{
  id: string,
  url: string,
  title?: string | null,
  publishedDate?: string | null,
  author?: string | null,
  score?: number,
  text?: string,
  highlights?: string[],
  highlightScores?: number[],
  summary?: string
}[]
```
</Accordion>
</AccordionGroup>
