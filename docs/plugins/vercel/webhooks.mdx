The Vercel plugin handles incoming webhooks. Point your Vercel project's webhook URL at your Corsair HTTP handler (see [Overview](/plugins/vercel/overview) for setup context).

<Info>
**New to Corsair?** See [webhooks](/concepts/webhooks) and [hooks](/concepts/hooks).
</Info>

## Webhook map

- `deployment`
  - `created` (`deploymentCreated`)
  - `succeeded` (`deploymentSucceeded`)
  - `error` (`deploymentError`)
- `project`
  - `created` (`projectCreated`)

## HTTP handler setup

```ts app/api/webhook/route.ts
import { processWebhook } from "corsair";
import { corsair } from "@/server/corsair";

export async function POST(request: Request) {
    const headers = Object.fromEntries(request.headers);
    const body = await request.json();
    const result = await processWebhook(corsair, headers, body);
    return result.response;
}
```

## Events

## Deployment

### Created

`deployment.created`

Triggered when a deployment is created.

**Payload**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `type` | `string` | Yes | `deployment.created` |
| `createdAt` | `number` | Yes | — |
| `teamId` | `string` | No | — |
| `payload` | `object` | Yes | — |

<AccordionGroup>
<Accordion title="payload full type">

```ts
{
  deployment: {
    id: string,
    name: string,
    url: string,
    target?: string,
    meta?: Record<string, string>
  }
}
```
</Accordion>
</AccordionGroup>

**`webhookHooks` example**

```ts
vercel({
    webhookHooks: {
        deploymentCreated: {
            before(ctx, args) {
                return { ctx, args };
            },
            after(ctx, response) {
            },
        },
    },
})
```
