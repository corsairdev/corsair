### aliasCreate

`events.aliasCreate`

Create an alias linking two distinct user IDs

**Risk:** `write`

```ts
await corsair.posthog.api.events.aliasCreate({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `distinct_id` | `string` | Yes | — |
| `alias` | `string` | Yes | — |
