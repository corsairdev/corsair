## Incident Notes

### create

`incidentNotes.create`

Add a note to an incident

**Risk:** `write`

```ts
await corsair.pagerduty.api.incidentNotes.create({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `incident_id` | `string` | Yes | — |
| `content` | `string` | Yes | — |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `note` | `object` | Yes | — |

<AccordionGroup>
<Accordion title="note full type">

```ts
{
  id: string,
  content?: string,
  created_at?: string,
  user?: {
    id: string,
    type: string,
    summary?: string,
    html_url?: string | null
  },
  channel?: {
    type?: string,
    name?: string
  }
}
```
</Accordion>
</AccordionGroup>
