## Ai App

### getOutputs

Get the outputs of an AI app for a transcript

**Risk:** `read`

```ts

```

| `transcriptId` | `string` | No | — |
| `appId` | `string` | No | — |
| `limit` | `number` | No | — |
| `skip` | `number` | No | — |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `apps` | `object` | No | — |

<AccordionGroup>
<Accordion title="apps full type">

  outputs?: {

    app_id?: string | null,
    created_at?: number | null,

</Accordion>
