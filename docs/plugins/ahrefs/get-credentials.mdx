## Authentication Method

- **[`api_key`](/concepts/api-key)** - Ahrefs API key

## API Key Setup

### Step 1: Get Your API Key

1. Log in to [Ahrefs](https://ahrefs.com)
2. Go to your profile → **API**
3. Copy your API key

**Storing Credentials:**

```bash
pnpm corsair setup --plugin=ahrefs api_key=your-api-key
```

<Info>
Ahrefs API access requires an active subscription with API access enabled. API usage is metered based on your plan.
</Info>

## Required Credentials Summary

| Credential | Required For | Where to Find |
|-----------|--------------|---------------|
| API Key | All API calls | Ahrefs account → API |

For general information about how Corsair handles authentication, see [Authentication](/concepts/auth).
