## Customers

### create

`customers.create`

Create a Dodo customer

**Risk:** `write`

```ts
await corsair.dodopayments.api.customers.create({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `name` | `string` | Yes | — |
| `email` | `string` | Yes | — |
| `phone_number` | `string` | No | — |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | — |
| `name` | `string` | No | — |
| `email` | `string` | No | — |
| `phone_number` | `string` | No | — |
| `created_at` | `string` | No | — |
