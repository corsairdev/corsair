Use **Tavily** through Corsair: one client, typed API calls, optional local DB sync.

**What you get:**

- 4 typed API operations
- 4 database entities synced for fast `.search()` / `.list()` queries

## Setup

<Steps>

<Step title="Install">
```bash
pnpm install @corsair-dev/tavily
```

</Step>

<Step title="Add the plugin">
<Tabs>
<Tab title="Solo">
```ts corsair.ts
import { createCorsair } from 'corsair';
import { tavily } from '@corsair-dev/tavily';

export const corsair = createCorsair({
	// ... other config options,
	multiTenancy: false,
    plugins: [tavily()],
});
```
</Tab>
<Tab title="Multi-Tenant">
```ts corsair.ts
import { createCorsair } from 'corsair';
import { tavily } from '@corsair-dev/tavily';

export const corsair = createCorsair({
	// ... other config options,
    multiTenancy: true,
    plugins: [tavily()],
});
```

See [Multi-tenancy](/concepts/multi-tenancy) for account isolation.

</Tab>
</Tabs>

</Step>

<Step title="Get credentials">
Follow [Get Credentials](/plugins/tavily/get-credentials) if you need help getting keys.

</Step>

<Step title="Store credentials">
<Tabs>
<Tab title="Solo">
```bash
pnpm corsair setup --plugin=tavily
```

Use the key names documented in [Get Credentials](/plugins/tavily/get-credentials) (for example `api_key=`, `bot_token=`, or OAuth client fields).
</Tab>
<Tab title="Multi-Tenant">
```bash
pnpm corsair setup --plugin=tavily --tenant=<tenantId>
```

Store per-tenant secrets after you create the tenant record. See [Multi-tenancy](/concepts/multi-tenancy).
</Tab>
</Tabs>

</Step>

</Steps>

## Authentication

Each tab shows how to register the plugin for that authentication method. The default `authType` from the plugin does not need to appear in the factory call.

<Tabs>
<Tab title="API Key (Default)">

```ts corsair.ts
tavily()
```

Store credentials with `pnpm corsair setup --plugin=tavily` (see [Get Credentials](/plugins/tavily/get-credentials) for field names). For OAuth, you typically store integration keys at the provider level and tokens per account or tenant.

More: [API Key](/concepts/api-key)

</Tab>
</Tabs>


## Query synced data

Synced entities support `corsair.tavily.db.<entity>.search()` and `.list()`. See [Database](/plugins/tavily/database) for filters and operators.


## Example API calls
