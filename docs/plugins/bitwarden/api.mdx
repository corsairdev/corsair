## Collections

### get

`collections.get`

Get details for a specific collection

**Risk:** `read`

```ts
await corsair.bitwarden.api.collections.get({});
```

**Input**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `organizationId` | `string` | Yes | — |
| `id` | `string` | Yes | — |



**Output**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `id` | `string` | Yes | — |
| `organizationId` | `string` | Yes | — |
| `name` | `string` | Yes | — |
| `externalId` | `string` | No | — |



---
